# Demo - Data Warehouse y Data Mart

La demo de Clase 08 existe para mostrar una idea muy concreta:

**no toda capa analitica tiene el mismo alcance ni el mismo consumidor.**

## Que muestra

- una necesidad analitica mas amplia e integrada
- una necesidad analitica mas especializada
- que datos integrarias en una capa central
- que parte servirias en una capa enfocada

## Que no intenta hacer

- construir una arquitectura empresarial completa
- enseñar una herramienta ETL
- resolver el proyecto final completo

El objetivo es pedagogico: comparar alcance, integracion, consistencia y consumo.
